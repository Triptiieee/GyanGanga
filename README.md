# Smart Student Onboarding Platform

A web-based prototype designed to simplify the student onboarding experience by providing a centralized dashboard for academic and administrative activities.

## Features
- Centralized dashboard with smart alerts and to-do list  
- Academic module with course information and branch-based views  
- Digital library system with issued books, due dates, and fine calculation  
- Railway concession application with real-time status  
- Events section for hackathons and workshops  

## Purpose
This project aims to reduce student confusion caused by fragmented systems and manual processes by bringing all essential student services into a single platform.

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## Note
This is a front-end prototype developed for a hackathon to demonstrate user experience and system workflow.
